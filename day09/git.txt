3. 아스키코드
	정의
		American Standard Code for Information Interchange, 미국 정보 교환 표준 부호
		영문 알파벳을 사용하는 문자 인코딩
		7비트 부호체계 (128) [0~127]
		8비트 부호체계 + 유럽어, 추가 특수문자 (256) [0~255]

	형태/모양 코드 예(대표적으로 2~3개)

		문자=10진수=16진수 
		"0" = 48 = 0X30

		문자=10진수=16진수
		NULL(공백)=0=0x00

	사용되는 곳
		컴퓨터나 통신장비에서 사용 *문자를 사용하는 장치*
		문자 인코딩은 아스키에 기초를 두고 있음
